Hartree Fock Self Consistent Field: A Simple Python Implementation

Basic Review

Hartree-Fock Equation

Hartree Fock Theory is a classic theory for solving many electron problem, which is useful in Chemistry, since molecules, nanoparticles and catalysts contain a lot of electrons. The theoretic background with detail deduction is referred to a very good book Modern Quantum Chemistry(Szabo). I will summarize the points here.

This theory approximates that the electrons move in the averaged field exerted from all the rest of electrons. Instead of using Hamilatonian to solve the schordinger equation:

HΨ=EΨH\Psi = E\Psi HΨ=EΨ

We replace the Hamiltonian with the Fock operator, or the Fockian:

Fψi=ϵiψiF\psi_i = \epsilon_i\psi_i Fψi​=ϵi​ψi​

Ψ\PsiΨ is a wavefunction of whole system, ψi\psi_iψi​ is a wavefunction of a single orbital. Then System’s wavefunction can be constructed from a list of occupied single orbital wavefunction via the Slater determinant.

Fockian contains the interactions that one electron will feels,

F=Ti⏟kinetic energy+∑RVnuc(R)⏟attraction from nuclear+∑jJ(j)−K(j)⏟ee repulsionF = \underbrace{T_i}_{\text{kinetic energy}} + \underbrace{\sum_{R}V_{nuc}(R)}_{\text{attraction from nuclear}} + \underbrace{\sum_j J(j) - K(j)}_{\text{ee repulsion}} F=kinetic energyTi​​​+attraction from nuclearR∑​Vnuc​(R)​​+ee repulsionj∑​J(j)−K(j)​​

The Self Consistent Field (SCF)

In order to construct the Fockian, information about all the electron should be known at first which is what we need to know at last. To solve this problem, a initial guess for the electron’s wavefunction {ψi0\psi^0_iψi0​} is needed. According to these wavefunction, a initial Fockian F0F^0F0 can be constructed, then a set of new wavefunctions can be solved {ψi1\psi_i^1ψi1​}, then a Fokian 1 F1F^1F1 is constructed, etc…

Matrix Form of Fockian: The Hartree-Fock-Roothan Equation

Usually we won’t present a wavefunction by a long set of data, instead we use the basis functions. Every single electron wavefunction can be expanded in terms of basis function:

ψi=∑μcμϕμ\psi_i = \sum_\mu c_\mu \phi_\mu ψi​=μ∑​cμ​ϕμ​

The basis function ϕμ\phi_\muϕμ​ won’t change during the calculation, but the coefficient {cμc_\mucμ​} does. Then the Fockian can be projected into the space of basis function, then become a matrix:

FC=Cϵ\textbf{F}\textbf{C}= \textbf{C}\epsilon FC=Cϵ

For the non-orthogonal basis

FC=SCϵ\textbf{F}\textbf{C}= \textbf{SC}\epsilon FC=SCϵ

we can orthogonalize the basis at first by find X=S−1/2\textbf{X}= \textbf{S}^{-1/2}X=S−1/2, then the following condition is fulfilled:

X†SX=I\textbf{X}^\dagger\textbf{S}\textbf{X} = \textbf{I} X†SX=I

Substitute XX−1\textbf{X}\textbf{X}^{-1}XX−1 into Fockian

X†FXX−1C=X†SXX−1Cϵ\textbf{X}^\dagger\textbf{FX}\textbf{X}^{-1}\textbf{C} = \textbf{X}^\dagger\textbf{SX}\textbf{X}^{-1}\textbf{C}\epsilon X†FXX−1C=X†SXX−1Cϵ

F′C′=C′ϵ\textbf{F}^\prime \textbf{C}^\prime = \textbf{C}^\prime\epsilon F′C′=C′ϵ

WhereX†FX=F′\textbf{X}^\dagger\textbf{FX} = \textbf{F}^\primeX†FX=F′, X−1C=C′\textbf{X}^{-1}\textbf{C} = \textbf{C}^\primeX−1C=C′

The Recipe

Follow the recipe from Modern Quantum Chemistry, we use the simple sto-3g basis.

  1. Give the system information: the element position, the element nuclear charge and the electron number of whole system. (main.py)
  2. Calculate the basic integration: kinetic integral, nuclear attraction integral, electron repulsion integral (integration.py)
  3. Obtain the X\textbf{X}X (math.py)
  4. Initial guess of a set of coefficient (i.e. a set of single electron wavefunction(main.py)
  5. Obtain the density matrix(integration.py)
  6. Obtain the fock and fock prime matirx( integration.py)
  7. Diagonize and solve fock prime to get C′\textbf{C}^\primeC′ (main.py)
  8. Get new C\textbf{C}C from C′\textbf{C}^\primeC′ (main.py)
  9. Compare the difference between old C\textbf{C}C and new C\textbf{C}C. (main.py)(I took the matrix difference and select out the max difference)
  10. Check whether the difference met the threshold requirement, if not, go back to fifth step (main.py)(you set threshold value)
